# 第四周心得小结
    这是参加训练营的第六周，也是修炼内功的第六周。主要学习了动态规划算法。以下进行总结：

动态规划：本质是 分治+最优子结构

动态规划的关键点：
    1.最优子结构 opt[n] = best_of(opt[n-1], opt[n-2],...)  
    2.储存中间状态 opt[i]  
    3.递推公式  
        例如：Fib: opt[i] = opt[i-1] + opt[i-2]  
         二维路径：opt[i, j] = opt[i+1, j] + opt[i, j+1](切判断a[i, j]是否是空地)  

思维总结：
1.打破自己的思维惯性，形成机器思维  
2.理解复杂逻辑的关键
